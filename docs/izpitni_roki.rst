izpitni_roki
============

V tem (komajda) modulu so v nekaj datotekah shranjene
pomožne in glavne funkcije, s katerimi naredimo ``html`` stran.

izpitni\_roki.osnovno
---------------------

V tej datoteki so definirani razni razredi in funkcije, ki jih potrebujemo
za lažje generiranje končne html strani.
